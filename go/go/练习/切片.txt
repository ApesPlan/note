Slice 定义
s1 := []int{2, 3, 4, 5}
s2 := make([]int, 16)
s3 := make([]int, 10, 32)
copy(s2, s1) // [2, 3, 4, 5, 0...]
s2 = append(s2[:3], s2[4:]...)
// 去头
s2 = s2[1:]
// 去尾
s2[:len(s2) - 1]

arr := [...]int{0, 1, 2, 3, 4, 5, 6, 7} // 数组
s1 := arr[2:6] // [2, 3, 4, 5]
s1[0] = 10
s2 := arr[3:5] // [5, 6]
Slice 本身没有数据，是对底层array的一个view
arr的值变为[0, 10, 2, 3, 4, 5, 6, 7]
